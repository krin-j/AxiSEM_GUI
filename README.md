# AxiSEM3D GUI
A graphical user interface for [AxiSEM3D](https://github.com/AxiSEMunity/AxiSEM3D), built with Qt 6.
> **This project is currently under active development and is not ready for download or use.**
## About
This GUI will provide a user-friendly interface for using AxiSEM3D, replacing the manual editing of YAML input files and command-line executions. It supports both local and remote (SSH) runs.
## Build Requirements
- Qt 6.5+
- C++17 compiler
- CMake 3.19+
